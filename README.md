The Channels DVR Server allows you to watch and record live TV using a web browser.

## About

This add-on runs Channels DVR Server, which allows you to manage live TV and recordings via the web browser. The add-on provides a single-click installation and run solution for Home Assistant.

Channels DVR Server is a flexible, standalone media server that lets you watch TV your way. It runs on your Home Assistant device - there's no cloud to worry about. Your TV shows and movies will always be available.

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the "Channels DVR Server" add-on
3. Start the add-on
4. Check the logs to see if everything went well
5. Click "OPEN WEB UI" to access the Channels DVR interface
6. Follow the setup wizard to configure your TV sources

## Configuration

This add-on has minimal configuration options and should work out of the box.
